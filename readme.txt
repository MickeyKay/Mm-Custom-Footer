Adds a custom MIGHTYminnow footer.

== Description ==
This plugin adds a custom MIGHTYminnow footer to say something like "Powered by MIGHTYminnow Website Weekend".

Configure the plugin via Settings > Mm Custom Footer.

== Installation ==

= Manual Installation =

1. Upload the entire `/mightyminnow-custom-footer` directory to the `/wp-content/plugins/` directory.
2. Activate MIGHTYminnow Custom Footer through the 'Plugins' menu in WordPress.
3. All set, no additional configuration needed. The custom footer should now appear.
